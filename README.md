# Online Food Ordering

## Description
It is basically Food Ordering Website where users can browse, order, and manage food items. Used here Integrated RESTful API to fetch real-time data of food items, including details, prices, and availability, ensuring users always see updated menu items. The website is Designed and implemented responsive frontend using HTML, CSS and JavaScript And Utilized PHP to handle server-side functionality, including session management, user authentication, and secure order processing.

## Features
- User login
- Admin panel
- Order management
- Payment status handling

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- Bootstrap

## How to Run
1. Clone the repository
2. Move project to htdocs
3. Start XAMPP
4. Open browser and run the project

---->http://localhost:8080/online_food/splash.php?parameter=diguu<----
